## 생성 우선순위

1. **필수 (1순위)**: 의료진 프로필 3장 + 시술 대표 6장 + 히어로 2장 = **11장**
2. **중요 (2순위)**: Before/After 12세트 24장 = **24장**
3. **보조 (3순위)**: 병원 시설 6장 + 의료진 활동 6장 = **12장**
4. **선택 (4순위)**: 장비 3장 + 분위기 3장 = **6장**
